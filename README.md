# 🎬 Movie Orbs

> Movies are more than entertainment—they are records of people, societies, and their times.

Movie Orbs는 사용자가 다양한 영화를 쉽고 직관적으로 탐색할 수 있도록 제작한 영화 정보 웹 애플리케이션입니다. 영화를 단순한 콘텐츠가 아닌 시대와 사회를 담아내는 문화적 기록물로 바라보며, 더 많은 작품을 발견하고 이해할 수 있는 경험을 제공하는 것을 목표로 개발했습니다.

---

## 📖 Project Motivation

영화는 현실 세계를 바탕으로 만들어진 예술입니다. 한 편의 영화에는 제작 당시의 사회 분위기, 문화, 가치관, 그리고 사람들이 살아가는 방식이 담겨 있습니다. 또 다른 세계를 통해 나를 알아보기 위해 이 애플리케이션을 제작했습니다.

---

## ✨ Features

### 🎥 Browse Popular Movies

현재 인기 있는 영화들을 한눈에 확인할 수 있습니다.

### 🔍 Search Movies

영화 제목을 검색하여 원하는 작품을 빠르게 찾을 수 있습니다.

### 📄 View Detailed Information

포스터, 평점, 개봉일 등 영화의 주요 정보를 확인할 수 있습니다.

### 📱 Responsive Design

다양한 기기 환경에서 편리하게 이용할 수 있도록 반응형 웹으로 구현했습니다.

## 🚀 Getting Started

### Clone Repository

```bash

cd Movie-Orbs
```

### Run Locally

프로젝트를 클론한 후 `index.html`을 실행하거나 VS Code의 Live Server를 이용해 실행할 수 있습니다.
